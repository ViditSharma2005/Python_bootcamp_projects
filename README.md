## What Each Part Is For

- `1_Python_Utilities/` contains small beginner Python scripts and practice files.
- `2_Data_Handling/` contains CSV, JSON, and file-handling practice projects.
- `libraries/` contains helper scripts and extra practice modules.
- `.env` stores private environment variables such as API keys and should not be committed.
- `.gitignore` keeps local-only files out of GitHub.
- `.venv/` is the local Python virtual environment.
